# レポート構成ガイド

## セクション詳細

### 1. 今月のハイライト
- 3〜5項目のトップニュース
- 箇条書きで簡潔に
- インパクトの大きい順に記載

```markdown
## 今月のハイライト

- **Apple × Google提携**: SiriにGeminiを採用する複数年パートナーシップを発表
- **Moltbot（旧Clawdbot）**: GitHub史上最速で急成長したAIエージェントが大バズり
- **ChatGPTへの広告導入**: OpenAIが収益多角化に向けた大きな一歩
```

### 2. 主要AIニュース・発表
- 企業別にセクション分け
- 表形式で日付・内容を記載
- 日付が不明な場合は「-」
- **オープンソース/コミュニティ**セクションを必ず含める

```markdown
### OpenAI / ChatGPT

| 日付 | 内容 |
|------|------|
| 1/16 | **ChatGPTへの広告導入発表** - 詳細説明 |
| 1/8 | **新機能リリース** - 詳細説明 |

### オープンソース / コミュニティ

| 日付 | 内容 |
|------|------|
| 1/27 | **Moltbot（旧Clawdbot）** - GitHub史上最速で80,000スター達成のAIエージェント |
| 1/15 | **LocalAI v2.0** - セルフホスト型LLMツールのメジャーアップデート |
```

### 3. 今月のリリースノート要点
- モデル/製品ごとにセクション分け
- 技術的な変更点を表形式で整理
- ベンチマークスコアがあれば記載

```markdown
### GPT-5.2 / GPT-5.2-Codex（OpenAI）

**リリース日**: 1/7

| カテゴリ | 主な改善点 |
|----------|-----------|
| **長期タスク対応** | Context Compactionにより文脈維持が改善 |
| **ベンチマーク** | SWE-Bench Pro, Terminal-Bench 2.0で最先端性能 |
```


### 5. 今月の数字
- 表形式で統計をまとめる
- 出典があれば括弧書きで記載

```markdown
| 指標 | 数値 |
|------|------|
| ChatGPT週間アクティブユーザー | 8億人 |
| ChatGPT市場シェア | 64.5%（前年比 -22pt） |
```

### 6. 来月の注目ポイント
- 番号付きリストで3〜5項目
- 予告、噂、注目イベントなど

```markdown
1. **Apple × Google提携の詳細発表**: 財務条件や技術的な詳細
2. **ChatGPT広告の影響**: ユーザー体験・収益への影響
```

### 7. 所感・メモ欄
- ユーザー記入用の空欄
- 箇条書きのプレースホルダーを用意

```markdown
## 所感・メモ欄

（チーム共有前に気になるトピックをここにメモ）

-
-
-
```

## 絵文字の使用

各セクションに適切な絵文字を付与:
- 今月のハイライト: 📌
- 主要AIニュース: 🔔
- リリースノート: 📋
- バズ・トレンド: 🔥
- 今月の数字: 📊
- 来月の注目: 🔮
- 所感: 💡
