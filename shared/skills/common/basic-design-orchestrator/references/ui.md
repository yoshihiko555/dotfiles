# UI（画面基本設計）テンプレ

## どちらのモードかを決める

- **単一画面モード（推奨）**: 1画面をMarkdown 1枚で完結させる
- **画面群モード**: 同一Markdown内で「画面一覧 → 各画面詳細」を書く

指定がなければ「単一画面モード」で進める。

## UIチェック（最低限）

- 画面の目的と入力項目が一致している
- 必須/制約/初期値が項目表に揃っている
- エラー時のユーザー体験（メッセージ/復旧手段）が書かれている
- 権限・表示条件が曖昧でない
