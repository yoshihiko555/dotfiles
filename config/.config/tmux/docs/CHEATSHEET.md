# tmux キーバインド チートシート

Prefix: `Ctrl+Q`

## ペイン操作

| キー | 動作 |
|------|------|
| `Prefix + r` | 横分割 |
| `Prefix + d` | 縦分割 |
| `Prefix + x` | ペイン閉じ |
| `Prefix + z` | ペインズーム (スナップショット保存付き) |
| `Prefix + h/j/k/l` | ペイン移動 |
| `Alt + h/j/k/l` | ペイン移動 (smart-splits, Prefix 不要) |

## ペインモード (`Prefix + p`)

モード中は Prefix 不要で連続操作可能。`Escape`/`Enter`/`q` で抜ける。

| キー | 動作 |
|------|------|
| `h/j/k/l` | ペイン移動 |
| `r` | 横分割 |
| `d` | 縦分割 |
| `x` | ペイン閉じ |
| `z` | ズーム |
| `1-5` | ペイン番号でジャンプ |
| `e` | リサイズモードに切替 |

## リサイズモード (`Prefix + e`)

| キー | 動作 |
|------|------|
| `h/j/k/l` | 5 セル単位でリサイズ |
| `H/J/K/L` | 1 セル単位でリサイズ |
| `Escape`/`Enter`/`q` | モード終了 |

## ウィンドウ

| キー | 動作 |
|------|------|
| `Prefix + c` | 新しいウィンドウ |
| `Prefix + n` | 次のウィンドウ |
| `Prefix + P` | 前のウィンドウ |

## セッション

| キー | 動作 |
|------|------|
| `Alt + 1-9` | N 番目のセッションに切替 (Prefix 不要) |
| `Prefix + s` | セッション一覧 (fzf) |

## ポップアップ (display-popup)

| キー | 動作 |
|------|------|
| `Prefix + g` | lazygit (80x80%) |
| `Prefix + t` | 一時シェル (80x80%) |
| `Prefix + C` | Claude Code (90x90%) |
| `Prefix + a` | AI サブエージェント履歴 (90x90%) |

## AI 監視 (ai-orchestra)

| キー | 動作 |
|------|------|
| `Prefix + A` | Claude 監視ペイン トグル |

## コピーモード

| キー | 動作 |
|------|------|
| `Prefix + [` | コピーモード開始 |
| `Prefix + V` | ペインスナップショット表示 (popup) |
| マウスドラッグ | 選択 → コピー (コピーモード維持) |

## 設定

| キー | 動作 |
|------|------|
| `Prefix + R` | 設定リロード |

## モード表示 (ステータスバー左端)

| バッジ | 状態 |
|--------|------|
| `COPY` (黄) | コピーモード中 |
| `PANE` (緑) | ペインモード中 |
| `RESIZE` (青) | リサイズモード中 |
