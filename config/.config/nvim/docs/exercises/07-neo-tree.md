# Exercise 07: Neo-tree ファイル操作
> 所要時間: 15-20分 | 前提: Exercise 01-03

## 目標
- Neo-tree でファイル/ディレクトリの作成・削除・リネーム・移動ができる
- ナビゲーションと表示切替を使いこなす
- 分割ウィンドウでファイルを開く

## キーリファレンス

| キー | 説明 |
|------|------|
| `<leader>e` | Neo-tree を開く/閉じる |
| `<leader>E` | 現在のファイルを Neo-tree で表示 |
| `Enter` / `o` | ファイルを開く / ディレクトリ展開 |
| `S` | 水平分割で開く |
| `s` | 垂直分割で開く |
| `a` | 新規作成 (末尾 `/` でディレクトリ) |
| `d` | 削除 |
| `r` | リネーム |
| `c` | コピー |
| `m` | 移動 |
| `H` | 隠しファイル表示 toggle |
| `z` | すべてのノードを閉じる |
| `q` | Neo-tree を閉じる |
| `?` | ヘルプ表示 |

## 練習 1: 基本ナビゲーション

1. `<leader>e` で Neo-tree を開く
2. `j`/`k` でファイルを上下に移動
3. ディレクトリ上で `Enter` → 展開/折りたたみ
4. `<BS>` (Backspace) で親ディレクトリへ戻る
5. `z` ですべてのノードを閉じる
6. `?` でヘルプを確認

## 練習 2: ファイルを開く

1. Neo-tree 内で任意のファイルを選択
2. `Enter` で通常のウィンドウに開く
3. `<leader>e` で Neo-tree に戻る
4. 別のファイルを `s` で垂直分割で開く
5. 別のファイルを `S` で水平分割で開く
6. `<leader>E` で現在編集中のファイルを Neo-tree 上で表示（reveal）

## 練習 3: ファイル操作

以下の操作を実際のファイルシステムで行おう（練習後に削除すること）。

1. `a` で新しいファイルを作成: `test-file.txt`
2. `a` で新しいディレクトリを作成: `test-dir/` (末尾に `/` を付ける)
3. `test-file.txt` を選択して `r` でリネーム: `renamed-file.txt`
4. `renamed-file.txt` を選択して `m` で `test-dir/` に移動
5. `test-dir/` を選択して `d` で削除（確認プロンプトで `y`）

## 練習 4: 表示切替

1. `H` で隠しファイル（`.gitignore` など）の表示をトグル
2. `/` でフィルター入力 → ファイル名で絞り込み → `Esc` で解除
3. `R` でツリーをリフレッシュ

## 練習 5: 効率的なワークフロー

以下のシナリオを試そう:

1. `<leader>ff` でファイルを検索して開く
2. `<leader>E` で Neo-tree 上にそのファイルを表示
3. 隣のファイルを `s` で分割して比較
4. `q` で Neo-tree を閉じる

## 確認テスト
- [ ] `<leader>e` で Neo-tree を開閉できる
- [ ] `a` でファイルとディレクトリを作成できる
- [ ] `r` でリネーム、`m` で移動、`d` で削除できる
- [ ] `s`/`S` で分割ウィンドウにファイルを開ける
- [ ] `<leader>E` で現在のファイルを Neo-tree 上に表示できる
