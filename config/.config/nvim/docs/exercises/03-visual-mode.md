# Exercise 03: Visual モード
> 所要時間: 15-20分 | 前提: Exercise 02

## 目標
- `v`, `V`, `Ctrl+v` の3種類の Visual モードを使い分ける
- 選択範囲に対する操作 (d/c/y/>/<) に慣れる
- `gv` と `o` を活用する

## キーリファレンス

| キー | 説明 |
|------|------|
| `v` | 文字単位選択 |
| `V` | 行単位選択 |
| `Ctrl+v` | 矩形選択 (Visual Block) |
| `d` | 選択範囲を削除 |
| `c` | 選択範囲を変更 |
| `y` | 選択範囲をコピー |
| `>` / `<` | インデント増/減 |
| `gv` | 直前の選択範囲を再選択 |
| `o` | 選択範囲の反対端へ移動 |
| `u` / `U` | 小文字/大文字に変換 |

## 練習 1: 文字単位選択 (v)

以下の文から特定の部分を選択して操作しよう。

```
The quick red cat jumps over the lazy dog.quick
```

タスク:
1. `quick` の `q` にカーソルを置き `v` → `d` の末尾まで `e` で選択 → `y` でコピー
2. 行末に移動して `p` でペースト
3. `brown fox` を `v` で選択して `c` → `red cat` に変更

## 練習 2: 行単位選択 (V)

以下のコードブロックで行を操作しよう。

```python
def greet():
    print("Hello")
    print("World")
    print("Foo")
    print("Bar")
    return True
```

タスク:
1. `print("Hello")` の行で `V` → `jj` で3行選択 → `d` で削除
2. `u` で戻す
3. `print("Hello")` の行で `V` → `jj` で3行選択 → `>` でインデント
4. `gv` で再選択 → `<` でインデントを戻す

## 練習 3: 矩形選択 (Ctrl+v)

以下のリストの先頭に番号を追加しよう。

```
apple
banana
cherry
date
elderberry
```

タスク:
1. `apple` の `a` にカーソルを置き `Ctrl+v` → `4j` で5行分選択
2. `I` (大文字) → `- ` と入力 → `Esc` → 全行の先頭に `- ` が追加される

次に、追加した `- ` を削除しよう:
1. 先頭の `-` にカーソルを置き `Ctrl+v` → `4j` → `l` で2文字分選択
2. `d` で削除

## 練習 4: 矩形選択で末尾に追加

以下の変数宣言の末尾にセミコロンを追加しよう。

```
let a = 1;
let b = 2;
let c = 3;
let d = 4;
```

タスク:
1. 1行目の末尾にカーソルを置き `Ctrl+v` → `3j` → `$` で各行末を選択
2. `A` (大文字) → `;` と入力 → `Esc` → 全行の末尾に `;` が追加される

## 練習 5: gv と o の活用

以下のテキストで `gv` と `o` を使おう。

```
AAAA bbbb CCCC
DDDD EEEE FFFF
GGGG HHHH IIII
```

タスク:
1. `BBBB` を `v` で選択 → `y` でコピー → カーソルが移動する
2. `gv` で直前の選択範囲を再選択 → `U` で大文字に（既に大文字なので `u` で小文字に）
3. 1行目の `A` で `v` → 3行目の `I` まで選択 → `o` を押す → カーソルが選択の反対端に移動

## 確認テスト
- [x] `v`, `V`, `Ctrl+v` の違いを説明できる
- [x] `Ctrl+v` + `I` で複数行の先頭に同時挿入できる
- [x] `Ctrl+v` + `A` で複数行の末尾に同時追記できる
- [x] `V` で行選択して `>` / `<` でインデント操作できる
- [x] `gv` で前の選択を素早く再選択できる
