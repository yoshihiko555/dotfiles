# Exercise 01: 基本編集
> 所要時間: 15-20分 | 前提: hjkl/w/b/e での移動ができること

## 目標
- `d`, `c`, `y`, `p` オペレータを使いこなす
- `.` (ドットリピート) で効率的に繰り返す
- `dd`, `D`, `cc`, `C` の違いを理解する

## キーリファレンス

| キー | 説明 |
|------|------|
| `dd` | 行削除 |
| `D` | カーソルから行末まで削除 |
| `cc` | 行を変更 (削除→Insert) |
| `C` | カーソルから行末まで変更 |
| `yy` | 行コピー |
| `p` / `P` | ペースト (後/前) |
| `x` | 1文字削除 |
| `u` | Undo |
| `.` | 直前の操作を繰り返す |

## 練習 1: 行の削除とペースト

以下のリストから不要な行を `dd` で削除し、`p` で順序を入れ替えよう。

目標: りんご → バナナ → ぶどう → みかん の順に並べ替える。

```
りんご
バナナ
ぶどう
みかん
```

手順:
1. `不要な行1` にカーソルを合わせて `dd` で削除
2. `不要な行2` にカーソルを合わせて `dd` で削除
3. `りんご` の行で `dd` → `ぶどう` の上で `P` (前にペースト)
4. 同様に `バナナ`, `みかん` を正しい位置に移動

## 練習 2: d/c/y + モーション

以下のコードを編集しよう。

```python
def compute_sum(price, quantity, tax_rate):
    subtotal = price * quantity
    tax = subtotal
    total = subtotal + tax
```

タスク:
1. `calculate_total` にカーソルを置き、`cw` で `compute_sum` に変更
2. `tax_rate` にカーソルを置き、`dw` で削除
3. `return total` の行で `yy` → `p` で行を複製
4. 変更を `u` で戻す

## 練習 3: D と C

以下の各行の `|` の位置にカーソルを置いて操作しよう。

```
この部分は残す |この部分を D で消す
この部分は残す 新しいテキストを入力してみました
```

タスク:
1. 1行目の `|` にカーソルを合わせ、`D` → カーソル以降が消える
2. `u` で戻す
3. 2行目の `|` にカーソルを合わせ、`C` → Insert モードになる → 新しいテキストを入力 → `Esc`

## 練習 4: ドットリピート

以下のリストの各行頭の `- ` を `TODO: ` に変更しよう。

```
TODO: タスクA
TODO: タスクB
TODO: タスクC
TODO: タスクD
TODO: タスクE
```

手順:
1. 1行目の `-` にカーソルを置き、`c2l` で `- ` を削除して Insert モードに入る
2. `TODO: ` と入力して `Esc`
3. 次の行の `-` に移動して `.` → 同じ変更が繰り返される
4. 残りの行も `j.` で繰り返す

- [x] `dw`, `cw`, `yw` をスムーズに使える
- [x] `D` と `dd` の違いを説明できる
- [x] `C` と `cc` の違いを説明できる
- [x] `.` で直前の編集を繰り返せる
