# Neovim Cheatsheet

**Leader: `Space`** | Theme: tokyonight (moon) | Plugin Manager: lazy.nvim

## Vim 基本操作

### モード切替

| キー | 切替先 | 説明 |
|------|--------|------|
| `i` | Insert | カーソル前に挿入 |
| `a` | Insert | カーソル後に挿入 |
| `I` | Insert | 行頭に挿入 |
| `A` | Insert | 行末に挿入 |
| `o` | Insert | 下に新行を作って挿入 |
| `O` | Insert | 上に新行を作って挿入 |
| `v` | Visual | 文字単位選択 |
| `V` | Visual Line | 行単位選択 |
| `Ctrl+v` | Visual Block | 矩形選択 |
| `R` | Replace | 上書きモード |
| `:` | Command | コマンドラインモード |
| `Esc` / `jj` | Normal | Normal モードに戻る |

### カーソル移動

| キー | 説明 |
|------|------|
| `h` `j` `k` `l` | 左 / 下 / 上 / 右 |
| `w` | 次の単語の先頭 |
| `W` | 次の WORD の先頭 (空白区切り) |
| `b` | 前の単語の先頭 |
| `B` | 前の WORD の先頭 |
| `e` | 単語の末尾 |
| `E` | WORD の末尾 |
| `0` | 行頭 |
| `^` | 行頭 (非空白) |
| `$` | 行末 |
| `gg` | ファイル先頭 |
| `G` | ファイル末尾 |
| `{数字}G` | 指定行へジャンプ |
| `{` / `}` | 前/次の空行 (段落移動) |
| `%` | 対応する括弧へジャンプ |
| `f{char}` | 行内で {char} へ前方ジャンプ |
| `F{char}` | 行内で {char} へ後方ジャンプ |
| `t{char}` | {char} の手前へ前方ジャンプ |
| `T{char}` | {char} の手前へ後方ジャンプ |
| `;` / `,` | f/t の繰り返し / 逆方向 |
| `Ctrl+d` | 半ページ下 |
| `Ctrl+u` | 半ページ上 |
| `Ctrl+f` | 1ページ下 |
| `Ctrl+b` | 1ページ上 |
| `H` / `M` / `L` | 画面の上/中/下 |
| `zz` | カーソル行を画面中央に |
| `zt` / `zb` | カーソル行を画面上/下に |

### 検索・ジャンプ

| キー | 説明 |
|------|------|
| `/{pattern}` | 前方検索 |
| `?{pattern}` | 後方検索 |
| `n` / `N` | 次/前の検索結果 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `Ctrl+o` | ジャンプリスト: 前の位置に戻る |
| `Ctrl+i` | ジャンプリスト: 次の位置に進む |
| `gd` | ローカル定義へジャンプ (LSP有効時はLSP定義) |
| `gf` | カーソル下のファイルパスを開く |

### 編集 (Normal モード)

| キー | 説明 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `D` | カーソルから行末まで削除 |
| `cc` | 行を変更 (削除 + Insert) |
| `C` | カーソルから行末まで変更 |
| `yy` | 行コピー (yank) |
| `Y` | 行コピー (yank) |
| `p` / `P` | ペースト (後/前) |
| `u` | Undo |
| `Ctrl+r` | Redo |
| `.` | 直前の操作を繰り返す |
| `J` | 下の行を結合 |
| `~` | 大文字/小文字トグル |
| `>>` / `<<` | インデント増/減 |
| `==` | 自動インデント |

### オペレータ + モーション/テキストオブジェクト

**オペレータ**: `d`(削除) `c`(変更) `y`(コピー) `>`(インデント) `<`(インデント戻す)

オペレータ + モーションの組み合わせ例:

| キー | 説明 |
|------|------|
| `dw` | 単語を削除 |
| `d$` / `D` | 行末まで削除 |
| `d0` | 行頭まで削除 |
| `dG` | ファイル末尾まで削除 |
| `cw` | 単語を変更 |
| `yw` | 単語をコピー |
| `>}` | 次の空行までインデント |

**テキストオブジェクト**: `i`(inner/内側) `a`(around/外側) + 対象

| キー | 説明 |
|------|------|
| `diw` | 単語を削除 (空白除く) |
| `daw` | 単語を削除 (空白含む) |
| `ci"` | `"..."` の中身を変更 |
| `ca"` | `"..."` 全体を変更 |
| `di(` / `dib` | `(...)` の中身を削除 |
| `da(` / `dab` | `(...)` 全体を削除 |
| `di{` / `diB` | `{...}` の中身を削除 |
| `dit` | HTML タグの中身を削除 |
| `cip` | 段落を変更 |
| `yap` | 段落をコピー |

### Visual モード操作

| キー | 説明 |
|------|------|
| `d` / `x` | 選択範囲を削除 |
| `c` | 選択範囲を変更 |
| `y` | 選択範囲をコピー |
| `>` / `<` | 選択範囲のインデント増/減 |
| `=` | 選択範囲を自動インデント |
| `u` / `U` | 選択範囲を小文字/大文字に |
| `J` | 選択行を結合 |
| `gv` | 直前の選択範囲を再選択 |
| `o` | 選択範囲の反対端へ移動 |

### マーク

| キー | 説明 |
|------|------|
| `m{a-z}` | ローカルマークを設定 |
| `m{A-Z}` | グローバルマークを設定 |
| `` `{mark} `` | マーク位置へジャンプ |
| `'{mark}` | マーク行頭へジャンプ |
| `` `. `` | 最後に変更した位置へジャンプ |
| `` `" `` | 最後にファイルを閉じた位置へジャンプ |

### レジスタ

| キー | 説明 |
|------|------|
| `"{reg}y` | 指定レジスタにコピー |
| `"{reg}p` | 指定レジスタからペースト |
| `"0p` | 最後にコピーしたものをペースト (削除混在回避) |
| `"+y` | システムクリップボードにコピー |
| `"+p` | システムクリップボードからペースト |
| `:reg` | レジスタ一覧を表示 |

### コマンドライン

| コマンド | 説明 |
|----------|------|
| `:w` | 保存 |
| `:q` / `:q!` | 終了 / 強制終了 |
| `:wq` / `ZZ` | 保存して終了 |
| `:{範囲}s/old/new/g` | 置換 (`%` で全行) |
| `:noh` | 検索ハイライト解除 |
| `:{数字}` | 指定行へジャンプ |
| `:!{cmd}` | 外部コマンド実行 |
| `:split` / `:vsplit` | 水平/垂直分割 |

## カスタムキーバインド

### 基本操作

| キー | モード | 説明 |
|------|--------|------|
| `jj` | Insert | Normal モードに戻る |
| `Esc` | Normal | 検索ハイライト解除 |

### ウィンドウ移動

| キー | 説明 |
|------|------|
| `Ctrl+h` | 左ウィンドウ |
| `Ctrl+j` | 下ウィンドウ |
| `Ctrl+k` | 上ウィンドウ |
| `Ctrl+l` | 右ウィンドウ |

### ウィンドウ操作 (`<leader>w` → `Ctrl+w` プロキシ)

which-key により `<leader>w` が `Ctrl+w` のプロキシとして動作:

| キー | 説明 |
|------|------|
| `<leader>ws` | 水平分割 (`:split`) |
| `<leader>wv` | 垂直分割 (`:vsplit`) |
| `<leader>wc` / `<leader>wq` | ウィンドウを閉じる |
| `<leader>wo` | 他のウィンドウをすべて閉じる |
| `<leader>ww` | 次のウィンドウへ移動 |
| `<leader>w=` | ウィンドウサイズを均等に |
| `<leader>w+` / `<leader>w-` | 高さを増減 |
| `<leader>w>` / `<leader>w<` | 幅を増減 |
| `<leader>wH/J/K/L` | ウィンドウを左/下/上/右に移動 |

### バッファ (`<leader>b`)

| キー | 説明 |
|------|------|
| `<leader>bn` | 次のバッファ |
| `<leader>bp` | 前のバッファ |
| `<leader>bd` | バッファ削除 |

### ファイル検索 FzfLua (`<leader>f`)

| キー | 説明 |
|------|------|
| `<leader>ff` | ファイル検索 |
| `<leader>fg` | Grep 検索 |
| `<leader>fb` | バッファ検索 |
| `<leader>fh` | Help 検索 |
| `<leader>fr` | 最近のファイル |
| `<leader>fd` | Diagnostics |
| `<leader>fs` | ドキュメントシンボル |
| `<leader>fw` | ワークスペースシンボル |

### Git (`<leader>g`)

| キー | 説明 |
|------|------|
| `<leader>gc` | Git commits |
| `<leader>gs` | Git status |

### LSP (`<leader>c` / `<leader>r` 系 — バッファアタッチ時のみ)

| キー | 説明 |
|------|------|
| `gd` | 定義へジャンプ |
| `gD` | 宣言へジャンプ |
| `gr` | 参照一覧 |
| `gi` | 実装へジャンプ |
| `K` | ホバー情報 |
| `<leader>rn` | リネーム |
| `<leader>ca` | コードアクション |
| `<leader>cd` | 行の Diagnostics |
| `[d` / `]d` | 前/次の Diagnostic |

**対応 LSP**: gopls, pyright, ts_ls

### Quickfix (`<leader>x`)

| キー | 説明 |
|------|------|
| `]q` / `[q` | 次/前の quickfix |
| `<leader>xq` | Quickfix 開く |
| `<leader>xc` | Quickfix 閉じる |
| `<leader>xl` | Location list 開く |
| `<leader>xL` | Location list 閉じる |
| `<leader>xd` | Diagnostics → quickfix |

### Markdown (`<leader>m`)

| キー | 説明 |
|------|------|
| `<leader>mp` | ブラウザプレビュー開始 |
| `<leader>mP` | ブラウザプレビュー停止 |
| `<leader>mr` | エディタ内レンダリング toggle |

### Git Hunk (`<leader>h` / gitsigns)

| キー | モード | 説明 |
|------|--------|------|
| `]c` / `[c` | Normal | 次/前の hunk へ移動 |
| `<leader>hs` | Normal/Visual | hunk をステージ |
| `<leader>hr` | Normal/Visual | hunk をリセット |
| `<leader>hS` | Normal | バッファ全体をステージ |
| `<leader>hR` | Normal | バッファ全体をリセット |
| `<leader>hp` | Normal | hunk プレビュー (ポップアップ) |
| `<leader>hi` | Normal | hunk プレビュー (インライン) |
| `<leader>hb` | Normal | 行の blame 表示 |
| `<leader>hd` | Normal | diff this |
| `ih` | Operator/Visual | hunk テキストオブジェクト |

### Git Toggle (`<leader>t`)

| キー | 説明 |
|------|------|
| `<leader>tb` | inline blame トグル |
| `<leader>tw` | word diff トグル |

### CodeDiff (diff viewer)

| コマンド | 説明 |
|----------|------|
| `:CodeDiff` | ファイルエクスプローラ（git status） |
| `:CodeDiff main` | main との差分 |
| `:CodeDiff main...` | PR風diff（merge-base） |
| `:CodeDiff file HEAD` | 現在ファイルをHEADと比較 |
| `:CodeDiff history` | コミット履歴 |

diff view 内のキー:

| キー | 説明 |
|------|------|
| `]c` / `[c` | 次/前の変更へ |
| `]f` / `[f` | 次/前のファイルへ |
| `t` | レイアウト切替 (side-by-side ↔ inline) |
| `do` / `dp` | 変更取得 / 変更送出 |
| `-` | ファイルのステージ切替 |
| `q` | diff を閉じる |
| `g?` | ヘルプ表示 |

### その他

| キー | 説明 |
|------|------|
| `<leader>?` | バッファローカルキーマップ表示 (which-key) |

## プラグイン内部操作

### Neo-tree (ファイルエクスプローラ)

`<leader>e` で開いた後、ツリーバッファ内で使えるキー:

**ナビゲーション**

| キー | 説明 |
|------|------|
| `j` / `k` | 上下移動 |
| `Enter` / `o` | ファイルを開く / ディレクトリ展開 |
| `S` | 水平分割で開く |
| `s` | 垂直分割で開く |
| `t` | 新しいタブで開く |
| `<BS>` (Backspace) | 親ディレクトリへ移動 |
| `P` | プレビュー表示 |

**ファイル操作**

| キー | 説明 |
|------|------|
| `a` | 新規作成 (末尾 `/` でディレクトリ) |
| `d` | 削除 |
| `r` | リネーム |
| `c` | コピー |
| `m` | 移動 |
| `y` | パスをコピー (yank) |
| `x` | カット |
| `p` | ペースト |

**表示切替**

| キー | 説明 |
|------|------|
| `H` | 隠しファイル表示 toggle |
| `R` | リフレッシュ |
| `/` | フィルター |
| `z` | すべてのノードを閉じる |
| `q` | Neo-tree を閉じる |
| `?` | ヘルプ表示 |

### FzfLua (ファジーファインダー)

検索ウィンドウ内で使えるキー:

**結果の操作 (Insert モード)**

| キー | 説明 |
|------|------|
| `Enter` | 選択を開く |
| `Ctrl+v` | 垂直分割で開く |
| `Ctrl+x` | 水平分割で開く |
| `Ctrl+t` | 新しいタブで開く |
| `Ctrl+j` / `Ctrl+k` | 結果リストを上下移動 |
| `Ctrl+n` / `Ctrl+p` | 結果リストを上下移動 |
| `Ctrl+d` / `Ctrl+u` | プレビューをスクロール |
| `Ctrl+q` | 結果を Quickfix に送る |
| `Tab` | 複数選択 toggle |
| `Shift+Tab` | 複数選択 toggle (逆方向) |
| `Ctrl+a` | すべて選択 |
| `Esc` / `Ctrl+c` | 検索を閉じる |

### Alpha ダッシュボード

起動画面で使えるキー:

| キー | 説明 |
|------|------|
| `e` | 新規ファイル |
| `f` | ファイル検索 |
| `r` | 最近のファイル |
| `g` | Grep 検索 |
| `x` | ファイルエクスプローラ |
| `i` | init.lua を編集 |
| `q` | 終了 |
