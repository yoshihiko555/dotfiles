# Phase 2: インデントガイド選定

## 背景

ネストが深いコードでインデントの対応関係を視覚的に把握したい。

## 候補

| | indent-blankline.nvim | mini.indentscope | indent-guides.nvim |
|---|---|---|---|
| Stars | ~4.2k | ~8.8k (mini.nvim) | ~0.3k |
| 機能 | 全行にガイド線 + スコープハイライト | 現在スコープのみアニメーション表示 | 全行にガイド線 |
| treesitter 連携 | あり（スコープ検出） | あり | なし |
| カスタマイズ性 | 高い（色・文字・除外設定） | 中程度 | 低い |
| 情報量 | 多い（定番） | 中程度 | 少ない |

## 比較

### indent-blankline.nvim
- 強み: 全行にガイド線を表示しつつ現在スコープを強調、定番で情報が多い
- 弱み: デフォルトの色が tokyonight moon と相性が悪い（要調整）

### mini.indentscope
- 強み: 現在スコープのみをアニメーション表示、軽量
- 弱み: 全行のガイド線は表示しない

## 検証結果

indent-blankline.nvim を導入。
- ガイド線の色をデフォルトから `#545c7e` に変更し tokyonight moon で視認性を確保
- スコープハイライトを `#65bcff`（水色）に設定
- 全行のインデントガイドと現在スコープの強調が動作することを確認

## 結論

indent-blankline.nvim を採用。全行ガイド + スコープハイライトの両方が使える。
tokyonight moon 用の色調整が必要だったが、設定で対応済み。
