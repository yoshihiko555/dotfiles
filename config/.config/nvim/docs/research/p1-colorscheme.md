# Phase 1: カラースキーム選定

## 背景

Neovim移行の最初のステップとして、見た目の基盤となるカラースキームを選定する。
lualineやtelescopeなど他プラグインのテーマにも影響するため、エコシステムとの相性が重要。

## 候補

| | tokyonight.nvim | catppuccin/nvim | kanagawa.nvim |
|---|---|---|---|
| **Stars** | ~7,800 | ~7,200 | ~5,900 |
| **バリエーション** | Storm, Moon, Night, Day | Latte(light), Frappe, Macchiato, Mocha(dark) | Wave, Dragon, Lotus(light) |
| **Treesitter** | o | o | o |
| **LSP診断** | o | o（severity別カスタマイズ可） | o（色ブレンドユーティリティ付き） |
| **Lualine** | 組み込みテーマあり | 組み込みテーマあり | 自動検出で対応 |
| **ターミナル連携** | 40+（WezTerm対応） | 200+（WezTerm対応） | 限定的（WezTerm対応） |
| **プラグイン対応数** | 80+ | 30+（トグル式） | 明確なリスト無し |
| **最終コミット** | 2025-11 | 2026-02 | 2025-10 |
| **メンテナンス** | 活発（folkeエコシステム） | 最も活発 | やや停滞気味（issue 90件） |

## 比較

### tokyonight.nvim
- **強み**: プラグイン対応数が最多（80+）。lazy.nvimとの自動検出あり。folke製なのでwhich-key, trouble等との相性が保証される
- **弱み**: 特に無し。バランスが良い

### catppuccin/nvim
- **強み**: 起動速度最速（~1ms）。エコシステムが最大で、WezTermを含む200+のアプリと統一テーマにできる。メンテナンスが最も活発
- **弱み**: プラグイン対応は個別トグル式で、対応していないプラグインは自分で設定が必要

### kanagawa.nvim
- **強み**: WCAG 2.1 AA準拠のアクセシビリティ。アーティスティックなデザイン
- **弱み**: メンテナンスがやや停滞。プラグイン対応の明確なリストが無い。issueが90件溜まっている
