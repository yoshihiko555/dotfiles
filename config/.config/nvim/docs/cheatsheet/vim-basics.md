# Vim 基本操作

## モード切替

| キー | 切替先 | 説明 |
|------|--------|------|
| `i` | Insert | カーソル前に挿入 |
| `a` | Insert | カーソル後に挿入 |
| `I` | Insert | 行頭に挿入 |
| `A` | Insert | 行末に挿入 |
| `o` | Insert | 下に新行を作って挿入 |
| `O` | Insert | 上に新行を作って挿入 |
| `v` | Visual | 文字単位選択 |
| `V` | Visual Line | 行単位選択 |
| `Ctrl+v` | Visual Block | 矩形選択 |
| `R` | Replace | 上書きモード |
| `:` | Command | コマンドラインモード |
| `Esc` / `jj` | Normal | Normal モードに戻る |

## カーソル移動

| キー | 説明 |
|------|------|
| `h` `j` `k` `l` | 左 / 下 / 上 / 右 |
| `w` | 次の単語の先頭 |
| `W` | 次の WORD の先頭 (空白区切り) |
| `b` | 前の単語の先頭 |
| `B` | 前の WORD の先頭 |
| `e` | 単語の末尾 |
| `E` | WORD の末尾 |
| `0` | 行頭 |
| `^` | 行頭 (非空白) |
| `$` | 行末 |
| `gg` | ファイル先頭 |
| `G` | ファイル末尾 |
| `{数字}G` | 指定行へジャンプ |
| `{` / `}` | 前/次の空行 (段落移動) |
| `%` | 対応する括弧へジャンプ |
| `f{char}` | 行内で {char} へ前方ジャンプ |
| `F{char}` | 行内で {char} へ後方ジャンプ |
| `t{char}` | {char} の手前へ前方ジャンプ |
| `T{char}` | {char} の手前へ後方ジャンプ |
| `;` / `,` | f/t の繰り返し / 逆方向 |
| `Ctrl+d` | 半ページ下 |
| `Ctrl+u` | 半ページ上 |
| `Ctrl+f` | 1ページ下 |
| `Ctrl+b` | 1ページ上 |
| `H` / `M` / `L` | 画面の上/中/下 |
| `zz` | カーソル行を画面中央に |
| `zt` / `zb` | カーソル行を画面上/下に |

## 検索・ジャンプ

| キー | 説明 |
|------|------|
| `/{pattern}` | 前方検索 |
| `?{pattern}` | 後方検索 |
| `n` / `N` | 次/前の検索結果 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `Ctrl+o` | ジャンプリスト: 前の位置に戻る |
| `Ctrl+i` | ジャンプリスト: 次の位置に進む |
| `gd` | ローカル定義へジャンプ (LSP有効時はLSP定義) |
| `gf` | カーソル下のファイルパスを開く |

## 編集 (Normal モード)

| キー | 説明 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `D` | カーソルから行末まで削除 |
| `cc` | 行を変更 (削除 + Insert) |
| `C` | カーソルから行末まで変更 |
| `yy` | 行コピー (yank) |
| `Y` | 行コピー (yank) |
| `p` / `P` | ペースト (後/前) |
| `u` | Undo |
| `Ctrl+r` | Redo |
| `.` | 直前の操作を繰り返す |
| `J` | 下の行を結合 |
| `~` | 大文字/小文字トグル |
| `>>` / `<<` | インデント増/減 |
| `==` | 自動インデント |

## コマンドライン

| コマンド | 説明 |
|----------|------|
| `:w` | 保存 |
| `:q` / `:q!` | 終了 / 強制終了 |
| `:wq` / `ZZ` | 保存して終了 |
| `:{範囲}s/old/new/g` | 置換 (`%` で全行) |
| `:noh` | 検索ハイライト解除 |
| `:{数字}` | 指定行へジャンプ |
| `:!{cmd}` | 外部コマンド実行 |
| `:split` / `:vsplit` | 水平/垂直分割 |
