# レジスタ / マーク

## マーク

| キー | 説明 |
|------|------|
| `m{a-z}` | ローカルマークを設定 |
| `m{A-Z}` | グローバルマークを設定 |
| `` `{mark} `` | マーク位置へジャンプ |
| `'{mark}` | マーク行頭へジャンプ |
| `` `. `` | 最後に変更した位置へジャンプ |
| `` `" `` | 最後にファイルを閉じた位置へジャンプ |

## レジスタ

| キー | 説明 |
|------|------|
| `"{reg}y` | 指定レジスタにコピー |
| `"{reg}p` | 指定レジスタからペースト |
| `"0p` | 最後にコピーしたものをペースト (削除混在回避) |
| `"+y` | システムクリップボードにコピー |
| `"+p` | システムクリップボードからペースト |
| `:reg` | レジスタ一覧を表示 |
