# オペレータ + テキストオブジェクト / Visual モード

## オペレータ + モーション

**オペレータ**: `d`(削除) `c`(変更) `y`(コピー) `>`(インデント) `<`(インデント戻す)

組み合わせ例:

| キー | 説明 |
|------|------|
| `dw` | 単語を削除 |
| `d$` / `D` | 行末まで削除 |
| `d0` | 行頭まで削除 |
| `dG` | ファイル末尾まで削除 |
| `cw` | 単語を変更 |
| `yw` | 単語をコピー |
| `>}` | 次の空行までインデント |

## テキストオブジェクト

`i`(inner/内側) `a`(around/外側) + 対象

| キー | 説明 |
|------|------|
| `diw` | 単語を削除 (空白除く) |
| `daw` | 単語を削除 (空白含む) |
| `ci"` | `"..."` の中身を変更 |
| `ca"` | `"..."` 全体を変更 |
| `di(` / `dib` | `(...)` の中身を削除 |
| `da(` / `dab` | `(...)` 全体を削除 |
| `di{` / `diB` | `{...}` の中身を削除 |
| `dit` | HTML タグの中身を削除 |
| `cip` | 段落を変更 |
| `yap` | 段落をコピー |

## Visual モード操作

| キー | 説明 |
|------|------|
| `d` / `x` | 選択範囲を削除 |
| `c` | 選択範囲を変更 |
| `y` | 選択範囲をコピー |
| `>` / `<` | 選択範囲のインデント増/減 |
| `=` | 選択範囲を自動インデント |
| `u` / `U` | 選択範囲を小文字/大文字に |
| `J` | 選択行を結合 |
| `gv` | 直前の選択範囲を再選択 |
| `o` | 選択範囲の反対端へ移動 |
